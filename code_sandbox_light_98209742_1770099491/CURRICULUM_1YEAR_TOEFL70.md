```
- Listening: 15-18점 (30점 만점)

```

```
시작: A1 (초급)
3개월: A2 (초중급)
6개월: A2+ (중급)
9개월: B1 (중고급)
12개월: B1+ (고급) → TOEFL iBT 70점
```

## ✍️ **Writing 자동 첨삭 시스템**

### **통합형 Writing (Integrated Task)**
```
형식:
1. 리딩 지문 읽기 (3분, 230-300단어)
2. 리스닝 강의 듣기 (2-3분)
3. 에세이 작성 (20분, 150-225단어)

평가 기준:
✅ 리딩 요점 정확히 파악
✅ 리스닝 요점 정확히 파악
✅ 대조 관계 명확히 서술
✅ 문법/철자 정확성
✅ 논리적 구성
```

### **독립형 Writing (Independent Task)**
```
형식:
1. 주제 제시
2. 에세이 작성 (30분, 300단어+)

평가 기준:
✅ 명확한 주장
✅ 구체적 예시 (2-3개)
✅ 논리적 전개
✅ 문법/어휘 다양성
✅ 결론의 적절성
```

### **자동 첨삭 기능:**
```
1. 즉시 피드백:
   - 문법 오류 (빨간 밑줄)
   - 철자 오류 (파란 밑줄)
   - 단어 선택 (노란 밑줄)

2. 점수 예측:
   - 5점 척도 (0-5점)
   - 각 항목별 점수
   - 종합 코멘트

3. 개선 제안:
   - 더 나은 단어 제안
   - 문장 구조 개선
   - 논리 흐름 보완

4. 샘플 비교:
   - 고득점 샘플 에세이
   - 자신의 에세이와 비교
```
