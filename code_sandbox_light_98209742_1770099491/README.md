# Level 5 - English Learning Platform

## 📚 프로젝트 개요

**Level 5**는 초등 4학년부터 예비 중1 학생들을 위한 체계적인 영어 학습 플랫폼입니다. 1년 커리큘럼(96회차)을 통해 TOEFL iBT 70점을 목표로 합니다.

---

## 🎯 학습 목표

### **1년 후 목표**
- **TOEFL iBT 70점**
  - Reading: 18-20점
  - Listening: 15-18점
  - Writing: 15-18점
  - Speaking: 별도 테이블 학습

### **난이도 진행**
- **시작**: 난이도 3/10 (초4-초5 수준)
- **종료**: 난이도 9/10 (중2-중3 수준)

---

## 📖 커리큘럼 구조

### **Level 5A (24주, 48회차)**
- **Week 1-8**: 기초 (100-200단어, TOEFL 5-8점)
- **Week 9-16**: 초중급 (200-300단어, TOEFL 8-12점)
- **Week 17-24**: 중급 (300-400단어, TOEFL 12-15점)
- **목표**: TOEFL Junior 준비

### **Level 5B (24주, 48회차)**
- **Week 1-8**: 중고급 (400-500단어, TOEFL 15-18점)
- **Week 9-16**: 고급 (500-600단어, TOEFL 18-20점)
- **Week 17-24**: 최고급 (600-700단어, TOEFL 20점+)
- **목표**: TOEFL iBT 70점

---

## 🏫 학습 흐름 (주 2회)

### **Step 1: Home Task (가정학습)**
등원 전 완료 필수

1. **AWL 어휘 학습** (30개)
   - 기초 10개 + 기초-중급 5개 + 중급 15개
   - 체크박스로 학습 진행 표시
   - **단어 클릭 시 발음 듣기** 🔊

2. **문법 워크시트**
   - Day 1: 관계사절 삽입구조
   - Pattern Hunting + Logical Error Detection
   - Brain-Mapping + 자동 채점

3. **명시적 문법 인강** (추후 탑재)
   - YouTube 영상 링크
   - 주차별 핵심 문법 개념

4. **Topic Builder**
   - 학술 영상 시청
   - 딕테이션 (16문항)
   - 한글 해설

---

### **Step 2: Booth (원내학습 45분)**
개별 독립 학습

1. **AWL 테스트** (필수 통과)
   - **35문제** (4지선다 30문제 + 스펠링 5문제)
   - 95점 이상 필수
   - 미통과 시 다음 단계 잠금
   - 스펠링: 발음 듣고 영어 단어 입력

2. **TOEFL Reading** (8분)
   - Day 1: 98단어 (Why Schools Need High Standards)
   - 관계사절 4개 포함 (문법 연계)
   - 찬성 입장
   - **탭 시스템**: 본문 읽기 / 문법 분석 / 핵심 어휘

3. **TOEFL Listening** (8분)
   - Day 1: 103단어 (The Problem with High Standards)
   - 관계사절 4개 포함
   - 반대 입장
   - TTS 오디오 재생
   - **딕테이션**: 8개 빈칸 채우기 (오디오 듣고 입력)
   - **탭 시스템**: 딕테이션 / 문법 분석

4. **논리구조도** (15분)
   - Reading vs Listening 대조 분석
   - 8 빈칸 채우기
   - 자동 채점 + 모범답안 비교

5. **Writing** (10분)
   - 5문장 에세이 작성
   - **유용한 표현 제공** (입장, 이유, 예시, 결론)
   - **초고급 자동 첨삭 필수** (100점 만점)
   - **제출 조건**: 첨삭 후 90점 이상 필수
     - 90점 미만: 제출 버튼 비활성화
     - 90점 이상: 제출 버튼 활성화
   - **첨삭 항목** (10개 항목):
     - 기본 형식 (8점): 대문자, 마침표, 문장 길이
     - 시제 일관성 (12점): 시제 혼용 방지, 3인칭 단수
     - 주어-동사 수 일치 (12점): 단수/복수 일치
     - 문장 형식 (8점): 주어+동사 완전한 문장
     - 타겟 문법: 관계사절 (15점): Day 1 문법 사용
     - 준동사 사용 (10점): to부정사, 동명사
     - 수동/능동 (10점): 적절한 태 사용
     - 어휘 사용 (10점): Day 1 AWL 어휘 5개 이상
     - **연결사 사용 (10점)**: 논리적 연결사 적절성
       - Sentence 1: I agree/believe/think
       - Sentence 2-3: First, Also, Another reason
       - Sentence 4: For example, In my experience
       - Sentence 5: Therefore, In conclusion
     - 구조 (5점): 5문장 완성

## 🎓 Day 1 상세 내용

### **AWL 어휘 30개**

#### **Group 1: 기초 (10개)**
important, student, learn, help, need, try, work, know, think, make

#### **Group 2: 기초-중급 (5개)**
goal, skill, future, prepare, problem

#### **Group 3: 중급 (15개)**
achieve, standard, challenge, support, improve, develop, experience, focus, understand, succeed, expect, require, benefit, progress, confident

---

### **문법: 관계사절 삽입구조**

#### **패턴**
```
관계사 + (주어 + 생각/전달 동사) + 진짜 동사
```

#### **예시**
- teachers **who experts believe** are good
- students **who teachers think** work hard
- schools **which people say** are excellent
- methods **that researchers believe** work best

---

### **Reading (찬성 입장)**

**주제**: Why Schools Need High Standards  
**단어 수**: 98단어  
**주장**: 높은 기준이 학생들의 학습과 미래 준비에 도움

**핵심 포인트**:
1. 높은 기준 → 더 많이 배움
2. 중요한 기술 개발 (문제 해결, 비판적 사고)
3. 인생에서 성공

---

### **Listening (반대 입장)**

**주제**: The Problem with High Standards  
**단어 수**: 103단어  
**반론**: 높은 기준이 학생들에게 문제 발생

**핵심 포인트**:
1. 학생들이 스트레스 받음
2. 자신감 상실
3. 학생별 지원 필요

---

### **논리구조도 (8 빈칸)**

| Reading (찬성) | Listening (반대) |
|---------------|----------------|
| Main Claim: standards | Counter-claim: problems |
| Supporting Point 1: learn | Problem 1: stressed |
| Supporting Point 2: skills | Problem 2: confidence |
| Conclusion: succeed | Alternative: support |

---

### **Writing 틀**

#### **찬성 틀**
```
Sentence 1: I [agree/believe/think] that schools [need/should have] high standards.
Sentence 2: First, high standards help students [learn more/develop skills].
Sentence 3: Also, students [who work hard] become [better learners/successful].
Sentence 4: For example, [when I study hard], I [learn more/achieve my goals].
Sentence 5: Therefore, high standards [are important/help students succeed].
```

#### **반대 틀**
```
Sentence 1: I [believe/think] that very high standards [can cause problems].
Sentence 2: First, high standards make students feel [stressed/worried].
Sentence 3: Another reason is that students may [lose confidence/stop trying].
Sentence 4: For example, some students [who try hard] still [feel like failures].
Sentence 5: Therefore, schools should [support each student/help at their level].
```
